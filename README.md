## 📝 Sobre o Projeto

O **SIGAH** é uma plataforma moderna e intuitiva desenvolvida para otimizar a administração escolar e o acompanhamento pedagógico de instituições de ensino. O ecossistema integra em um único lugar o gerenciamento de turmas, alunos, professores e disciplinas, permitindo uma comunicação fluida e relatórios precisos em tempo real.

O protótipo inicial da inteligência de negócios foi desenhado utilizando o **Google AI Studio** e evoluído para uma aplicação web robusta de ponta a ponta.

## 🚀 Principais Funcionalidades

* **Painel Administrativo (Dashboard):** Visão analítica com indicadores de desempenho e dados gerais da instituição.
* **Gestão de Disciplinas:** Controle de matriz curricular integrado às áreas de especialidade do corpo docente.
* **Gestão de Usuários:** Cadastro estruturado de Alunos, Professores e Funcionários com controle de status.
* **Matrículas e Turmas:** Vinculação inteligente de alunos a turmas e disciplinas correspondentes.
* **Relatórios Automatizados:** Exportação de listas e relatórios institucionais diretamente para planilhas Excel (`.xlsx`).
* **Design Responsivo:** Interface otimizada para desktops, tablets e dispositivos móveis desenvolvida com componentes Shadcn UI.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias mais modernas do ecossistema de desenvolvimento web:

* **Frontend:** [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/) + [Shadcn UI](https://ui.shadcn.com/)
* **Banco de Dados & Autenticação:** [Supabase (PostgreSQL)](https://supabase.com/)
* **Ícones:** [Lucide React](https://lucide.dev/)
* **Exportação de Dados:** [SheetJS (XLSX)](https://sheetjs.com/)
